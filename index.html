Tips, Hacks & Tricks

VSCode Hacks

  • Zen Mode: Press `Ctrl + K Z` (Windows/Linux) or `Cmd + K Z` (Mac) to enter Zen Mode.
  • Multi-Cursor Magic: Hold `Alt` (Windows/Linux) or `Option` (Mac) and click in multiple places where you want to place cursors.
  • Quick Rename: Highlight the variable and press `F2`, then enter the new name and press `Enter`.
  • Command Palette: Press `Ctrl + Shift + P` (Windows/Linux) or `Cmd + Shift + P` (Mac) to open the Command Palette.
  • Integrated Terminal: Press `Ctrl + `` (Windows/Linux) or `Cmd + `` (Mac) to open/close the integrated terminal.
  • Live Server: Install the "Live Server" extension from the VSCode marketplace. Right-click on your HTML file and select "Open with Live Server."

Git & GitHub Hacks

  • Speedy Commits: Run `git commit -am "your message"` to add and commit changes in one go.
  • Undo Last Commit: Run `git reset --soft HEAD~1` to undo the last commit.
  • Keep Your Fork Up to Date: Synchronize your fork with the upstream repository using `git fetch upstream` and `git merge upstream/master`.
  • GitHub CLI: Manage GitHub repositories from your terminal with commands like `gh repo create`, `gh issue list`, etc.
