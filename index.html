VSCode Hacks

  • Zen Mode: Press `Ctrl + K Z` (Windows/Linux) or `Cmd + K Z` (Mac) to enter Zen Mode. Press the same combination to exit.
  • Multi-Cursor Magic: Hold `Alt` (Windows/Linux) or `Option` (Mac) and click in multiple places to place cursors.
  • Quick Rename: Highlight the variable and press `F2` to rename all instances.
  • Command Palette: Press `Ctrl + Shift + P` (Windows/Linux) or `Cmd + Shift + P` (Mac) to open.
  • Integrated Terminal: Press `Ctrl + `` (Windows/Linux) or `Cmd + `` (Mac) to open/close the terminal.
  • Live Server: Install the "Live Server" extension, right-click on your HTML file, and select "Open with Live Server."

Git & GitHub Hacks

  • Speedy Commits: Run `git commit -am "your message"` to add and commit changes in one go.
  • Undo Last Commit: Run `git reset --soft HEAD~1` to undo the last commit.
  • Keep Your Fork Up to Date: Add upstream: `git remote add upstream `, fetch: `git fetch upstream`, and merge: `git merge upstream/master`.
  • Interactive Rebase: Run `git rebase -i HEAD~n` where `n` is the number of commits to rebase.
  • GitHub CLI: Install GitHub CLI with `brew install gh`, authenticate with `gh auth login`, and use commands like `gh repo create`.
